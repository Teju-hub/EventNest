# EventNest

A local event booking web application built with Node.js, Express, MySQL, and EJS.

## Setup

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with required credentials (see `.env.example`)
4. Start XAMPP and run MySQL on port 3307
5. Run `node server.js`
6. Visit `http://localhost:3003/eventnest`
